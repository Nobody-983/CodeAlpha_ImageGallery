# 📸 CodeAlpha Image Gallery

A responsive and interactive image gallery built using **HTML, Tailwind CSS, and JavaScript**.  
This project was developed as part of the CodeAlpha Frontend Development Internship.

---

## 🚀 Features

- 📱 Fully responsive design (mobile, tablet, desktop)
- 🖼️ Image gallery with uniform layout
- 🔍 Category filtering (All, Nature, Cars, Food)
- 📲 Mobile-friendly carousel (Next / Previous navigation)
- 🎨 Modern UI with Tailwind CSS styling
- ⚡ Smooth hover effects and transitions
- 🧠 Dynamic DOM manipulation using JavaScript

---

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript (Vanilla JS)

---

## 📱 Responsive Design

- **Mobile (< 500px):** Shows one image at a time with navigation buttons
- **Tablet (≥ 500px):** Grid layout with 2 columns
- **Desktop (≥ 768px):** Grid layout with 3 columns
