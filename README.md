A responsive image gallery web application built using HTML, Tailwind CSS, and JavaScript. The project allows users to view and filter images based on categories such as Nature, Cars, and Food. It also includes a mobile-friendly navigation system where users can browse images one at a time using Next and Previous buttons.

Features

Responsive grid layout for different screen sizes
Image filtering by category (All, Nature, Cars, Food)
Mobile-friendly single image view with navigation buttons
Smooth hover effects for better user experience
Clean and modern user interface

Technologies Used

HTML
Tailwind CSS
JavaScript
